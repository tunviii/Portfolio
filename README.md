# 🌐 Personal Portfolio

A modern, responsive developer portfolio showcasing my projects, technical skills, and experience in Full Stack Development and Artificial Intelligence.

Designed with a minimalist aesthetic, subtle animations, and an emphasis on clean user experience.

---

## ✨ Features

- Responsive design
- Smooth page transitions using Framer Motion
- Interactive project visualizers
- Professional resume section
- Experience timeline
- Modern UI with soft pastel color palette
- Optimized for desktop, tablet, and mobile

---

## ⚙️ Getting Started

Clone the repository

```bash
git clone https://github.com/tunviii/portfolio.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```
